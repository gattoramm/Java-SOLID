Приложение, которое нарушает принцип ISP

Класс I используется классами A и B.
В классе A используется один из методов класса I.
В классе B используется один из методов класса I.
Как только нужно изменить метод, используемый в классе A, нужно изменить этот метод в классе I.
Это приведет к тому, что нужно перекомпилировать класс I. А вместе с этим перекомпилируется класс B.
