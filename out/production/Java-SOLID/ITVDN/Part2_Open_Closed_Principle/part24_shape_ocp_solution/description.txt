Приложение, которое поддерживает принцип OCP

Разработаем калькулятор площади для одной из фигур.

Shape - абстрактный класс.

Rectangle - класс, отвечающий за создание прямоугольника.
Circle - класс, отвечающий за создание круга.
В каждом классе метод area() возвращает площадь.
метод areaCalc() в классе AreaCalculator принимает любой экземпляр класса, который
наследуется от абстрактного класса Shape
