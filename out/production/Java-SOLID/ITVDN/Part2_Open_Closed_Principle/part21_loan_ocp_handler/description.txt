Приложение, которое нарушает принцип OCP

Разработаем систему одобрения личных займов с проверкой.

LoanApprovalHandler - класс, отвечающий за одобрение займа.
approvePersonalLoan - метод, одобряющий займ

PersonalLoanValidator - класс проверки информации

VehicleLoanValidator - класс, отвечающий за одобрение покупки машины
approveVehicleLoan - метод принятия решения по займу для покупки машины

Проблема - наружение принципа SRP, так как каждый метод предназначен для определенный экземпляр класса
